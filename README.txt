O nosso projeto inclui o uso de uma placa ESP-32 para poder se conectar via WI-FI, uma celula de carga em conjunto com um conversor hx711, e um sensor de distancia 
ultrassonico.A celula de carga será usada para medir o peso dos resuiduos acumulados, e o sensor de distancia irá medir o volume, comparando a distancia deles até o sensor.
O front end consiste no nosso site para divulgar o produto e também disponibilizar informações sobre quantidade de lixo coletado, e também o app que irá monitorar o clean
drain do usuario.
Estamos desenvolvendo o back end para conseguir coletar as informações obtidas pela ESP-32 para envia-las para o app do usuario e apartir dele enviar para a internet
assim podendo enviar para o nosso site.
Dependemos de uma blibioteca compativel com o hx711 para a celula de carga e também do fiware.
